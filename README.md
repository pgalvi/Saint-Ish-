# Saint Ish | Photography Portfolio

Portafolio de Fotografía de Alta Gama. Editorial, Retratos y Campañas.

## 🚀 Estructura del Proyecto

*   `index.html` - Página principal y Galería Masonry.
*   `styles.css` - Estilos de diseño, animaciones y diseño responsivo.
*   `script.js` - Funcionalidades interactivas (Menú móvil, Lightbox, Observadores de sección).
*   `assets/` - Contiene las imágenes y recursos visuales de la galería.

## 💻 Ejecución Local

Para visualizar el proyecto localmente, puedes usar cualquier servidor estático. Si tienes **Node.js**:

```bash
npx http-server -p 8080 -c-1
```

Y visita `http://localhost:8080` en tu navegador.
